# ProofOfStake-Java

A simple academic implementation of a Proof-of-Stake (PoS) blockchain built in Java.

---

##  Project Overview

This project demonstrates the core concepts of a Proof-of-Stake blockchain system including:

- Block creation
- SHA-256 hashing
- Digital signatures
- Validator-based block validation
- Genesis block
- Chain validation mechanism

---

##  Technologies Used

- Java 17
- SHA-256 Cryptographic Hashing
- Java Security (Digital Signatures)
- Object-Oriented Programming

---

##  Project Structure

```
src/com/pos/blockchain/
│
├── Block.java
├── Blockchain.java
├── Validator.java
├── POSSelector.java
├── Transaction.java
└── Main.java
```

---

##  Features

✔ Genesis Block  
✔ Validator-based block creation  
✔ Proof-of-Stake selection  
✔ Digital signature verification  
✔ Blockchain integrity validation  

---

##  How to Run

### Compile

```
javac -d out src/com/pos/blockchain/*.java
```

### Run

```
java -cp out com.pos.blockchain.Main
```
